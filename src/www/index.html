About

random-host.com is the personal blog of web developer, wannabe dingo, sandbox game enthusiast and blogger Ch'Ih-Yu.

This sub-domain hosts the service endpoint for the custom Alexa skill centered around the author and the services hosted on this server such as game servers.

The source code is released under an open source license and can be found on GitHub.
